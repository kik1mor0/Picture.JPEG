# 🧩 Polyglot File: exploit.png

Демонстрация **полиглот-файла**, который одновременно является корректным PNG-изображением и исполняемым BAT-скриптом.

---

## 📦 Внешний вид и содержимое

| Для пользователя | Внутреннее содержимое | При запуске |
|------------------|------------------------|-------------|
| 🖼️ Картинка (.png) | ⚙️ Исполняемый скрипт (.bat) | Выполнение скрипта в `cmd.exe` |

---

## 🧬 Структура файла

| Байты | Содержимое | Описание |
|-------|------------|----------|
| **0–7** | `89 50 4E 47 0D 0A 1A 0A` | Магическая PNG сигнатура |
| **8–20** | `[PNG структура]` | Минимальные PNG данные (IHDR, и т.д.) |
| **21–N** | `@echo off`<br/>`echo Hello from hidden script!`<br/>`pause` | BAT-команды |
| **Расширение** | `.png` | Но содержит исполняемый код |

## ⚙️ Принцип работы

1. **Как PNG**  
   Любой просмотрщик изображений читает первые байты, видит корректную сигнатуру и отображает картинку. Остальные данные игнорируются.

2. **Как BAT**  
   При запуске через командную строку (или двойным кликом, если файл ассоциирован с `cmd.exe`) интерпретатор выполняет команды, начиная с `@echo off`. Бинарный PNG-заголовок не мешает выполнению.

---

## 🔧 Пример создания

Создать такой файл можно с помощью команды `copy` в Windows:

```cmd
copy /b image.png + script.bat exploit.png
```

Где:
- `image.png` – минимальное PNG-изображение (например, 1×1 пиксель);
- `script.bat` – BAT-скрипт (обязательно начинается с `@echo off` или другой команды, которая не нарушит синтаксис);
- `exploit.png` – итоговый полиглот-файл.
