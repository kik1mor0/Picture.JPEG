# Polyglot Files Generator

Генерация полиглот-файлов, которые одновременно являются корректными PNG-изображениями и исполняемыми скриптами (BAT/CMD) или программами (EXE).

---

## Что такое полиглот-файл?

Полиглот — файл, который распознаётся как разные форматы в зависимости от способа открытия. Например, один файл может:

- отображаться как обычная картинка в графическом редакторе;
- при запуске через командную строку выполнять BAT-скрипт или EXE-программу.

Это достигается за счёт специальной структуры файла: данные одного формата встраиваются в другой так, что оба типа приложений корректно интерпретируют файл.

---

## Содержимое репозитория

| Файл             | Назначение                                                                  |
|------------------|-----------------------------------------------------------------------------|
| `polyglot.py`    | Автоматическое создание полиглота PNG/BAT.                                  |
| `make_hybrid.py` | Создание гибридного файла PNG/EXE (исполняемый файл со встроенным изображением). |
| `hello.c`        | Исходный код на C для гибридного EXE (выводит «привет» в консоли).          |
| `example.png`    | Пример PNG-изображения для тестирования (опционально).                      |

---

## Требования

**Для PNG/BAT полиглота:**
- Python 3.6+ (только стандартная библиотека)

**Для PNG/EXE гибрида:**
- Python 3.6+ с пакетами `pefile` и `Pillow`:
  ```bash
  pip install pefile pillow
  ```
- Компилятор C (рекомендуется MinGW-w64 через MSYS2, `gcc` должен быть доступен в PATH)

## Использование

### 1. PNG/BAT полиглот

Создаёт файл, который открывается как PNG и выполняется как BAT-скрипт.

```bash
python polyglot.py input.png output.bat
```

По умолчанию скрипт выполняет:

```batch
echo привет
pause
```

Чтобы изменить команды, отредактируйте переменную `commands` в `polyglot.py`.

**Результат:** файл `output.bat`
- переименуйте в `.png` — откроется как изображение;
- запустите с расширением `.bat` — выполнится скрипт.

---

### 2. PNG/BAT полиглот через утилиту `pngtext` (альтернативный способ)

Если установлена утилита `pngtext` из пакета `libpng`:

```bash
pngtext -a -k cmd -t "goto :end" input.png temp.png
copy /b temp.png + script.bat polyglot.bat
```

где `script.bat` содержит нужные команды (`echo привет`, `pause` и т. д.).

---

### 3. PNG/EXE гибрид

Создаёт файл, который запускается как EXE (даже с расширением `.png`) и отображается как изображение в просмотрщике.

**Шаг 1.** Скомпилируйте программу из `hello.c`:

```bash
gcc -mwindows -o hello.exe hello.c
```

**Шаг 2.** Создайте гибридный файл:

```bash
python make_hybrid.py hello.exe image.png hybrid.png
```

**Результат:** файл `hybrid.png`
- двойной клик в Проводнике Windows — запустится программа (консоль с надписью «привет»);
- открытие в Paint или «Фотографиях» — отобразится исходная картинка.

> **Примечание:** некоторые просмотрщики отображают только файлы, начинающиеся с PNG-сигнатуры. В таком случае откройте файл в Paint или стандартном приложении «Фотографии» (Windows 10/11).

## Примеры

```bash
# Создание полиглота PNG/BAT
python polyglot.py my_image.png hello.bat

# Создание гибрида PNG/EXE
gcc -mwindows -o hello.exe hello.c
python make_hybrid.py hello.exe logo.png start.png
```

## ⚠️ Предостережения

**Безопасность.** Полиглоты могут использоваться для скрытого запуска вредоносного кода. Не запускайте подобные файлы из непроверенных источников. Антивирусы могут распознавать гибридные файлы как потенциально опасные.

**Совместимость.** Не все просмотрщики корректно открывают файлы, не начинающиеся с PNG-сигнатуры. Браузеры и некоторые сторонние редакторы могут отказаться открывать гибрид PNG/EXE.

**Только для Windows.** Оба метода (BAT и EXE) ориентированы на Windows. На Linux и macOS файлы не будут исполняться.

**Расширение файла.** Для PNG/BAT при запуске требуется расширение `.bat`. Для гибрида PNG/EXE расширение `.png` сохраняется, но файл запускается как EXE.
