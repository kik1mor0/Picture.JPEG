| Смещение (байты) | Содержимое | Описание |

| `8-20` | PNG структура (IHDR) | Минимальные данные для корректного отображения |

## 🔧 Принцип работы

1. **PNG-просмотрщик** читает первые байты, видит корректную сигнатуру и отображает картинку.
2. **Командная строка** (или двойной клик) интерпретирует содержимое как пакетный файл, игнорируя бинарный PNG-заголовок благодаря особенностям обработки `cmd.exe`.
